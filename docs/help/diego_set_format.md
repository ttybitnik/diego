## diego set format

Set output format for the Hugo data file (default "yaml")

### Synopsis

Set output format for the Hugo data file (default "yaml").

```
diego set format {yaml|json|toml|xml} [flags]
```

### Examples

```
diego set format yaml
diego s format json
```
