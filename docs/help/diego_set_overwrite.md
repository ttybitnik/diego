## diego set overwrite

Enable or disable the overwrite flag by default

### Synopsis

Enable or disable the overwrite flag by default.

```
diego set overwrite {true|false|1|0|enabled|disabled} [flags]
```

### Examples

```
diego set overwrite true
diego s overwrite false
```
