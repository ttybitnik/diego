## diego set shortcode

Enable or disable the shortcode flag by default

### Synopsis

Enable or disable the shortcode flag by default.

```
diego set shortcode {true|false|1|0|enabled|disabled} [flags]
```

### Examples

```
diego set shortcode true
diego s shortcode false
```
